## Contributing

When proposing changes to a protocol:

1. Create a draft spec: `{protocol}-v{X.Y}-draft.md`
2. Open an issue linking to the draft
3. After consensus, rename to stable version
4. Update this README

See [ADR-001](../adr/001-anp.md) for the architectural context behind ANP.
